### Mục tiêu tuần 5:

* Tìm hiểu cách triển khai một ứng dụng web đơn giản lên Amazon EC2.

### Kết quả đạt được tuần 5:

* Hiểu quy trình cơ bản để triển khai một web application lên Amazon EC2.
* Chuẩn bị được EC2 instance làm application server.
* Cài đặt môi trường runtime và các dependency cần thiết cho ứng dụng.
* Cấu hình inbound rules trong Security Group cho SSH, HTTP, HTTPS và port ứng dụng khi cần.
* Upload source code lên server bằng Git hoặc VS Code Remote - SSH.
* Khởi chạy ứng dụng trên EC2 và kiểm tra truy cập thành công từ trình duyệt.
* Hiểu cách DNS ánh xạ domain name đến địa chỉ server.
* Nắm vai trò cơ bản của Amazon Route 53 trong quản lý domain và DNS.
* Có thêm kinh nghiệm thực hành kết hợp compute, networking và triển khai ứng dụng.
