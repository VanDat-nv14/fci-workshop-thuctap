### Week 4 Objectives:

* Learn storage services on AWS with Amazon S3 and AWS Storage Gateway.
* Deploy a static website using S3 Static Website Hosting.
* Integrate CloudFront and explore versioning, replication, and resource cleanup.

### Week 4 Achievements:

* Created and configured S3 buckets for storing files and website assets.
* Practiced uploading, organizing, and managing objects in Amazon S3.
* Understood how Storage Gateway can connect on-premises or EC2-based workloads with S3 storage.
* Configured file shares to access data through Storage Gateway.
* Deployed a static website using S3 Static Website Hosting.
* Configured bucket and object permissions carefully to allow public website access when required.
* Tested the deployed website through the S3 website endpoint.
* Integrated CloudFront to improve content delivery performance.
* Learned how Bucket Versioning helps track and recover object changes.
* Practiced object movement and multi-region replication concepts.
* Cleaned up lab resources to reduce the risk of unexpected AWS charges.
