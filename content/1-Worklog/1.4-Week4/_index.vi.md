### Mục tiêu tuần 4:

- Tìm hiểu các dịch vụ lưu trữ trên AWS với Amazon S3 và AWS Storage Gateway.

### Kết quả đạt được tuần 4:

- Tạo và cấu hình được S3 bucket để lưu trữ file và tài nguyên website.
- Thực hành upload, sắp xếp và quản lý object trong Amazon S3.
- Hiểu cách Storage Gateway kết nối workload on-premises hoặc EC2 với lưu trữ S3.
- Cấu hình File Shares để truy cập dữ liệu thông qua Storage Gateway.
- Triển khai thành công website tĩnh bằng S3 Static Website Hosting.
- Cấu hình quyền bucket và object cẩn thận để cho phép public website khi cần thiết.
- Kiểm tra website thông qua S3 website endpoint sau khi deploy.
- Tích hợp CloudFront để cải thiện tốc độ phân phối nội dung.
- Hiểu cách Bucket Versioning hỗ trợ theo dõi và khôi phục các thay đổi của object.
- Thực hành khái niệm di chuyển object và replication đa vùng.
- Dọn dẹp tài nguyên lab để giảm rủi ro phát sinh chi phí ngoài ý muốn.
