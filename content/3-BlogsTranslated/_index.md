Below is the list of technical blogs selected and translated during my internship at AWS Vietnam. These articles have been summarized and translated to share valuable cloud knowledge with the engineering community.

### Table of Translated Blogs:

1. **[Blog 1 – Scaling Telco Automation with Red Hat AAP and ROSA on AWS](3.1-TelcoAnsible/)**
   * *Overview:* Explores automation architectures for firmware updates across millions of telecommunication endpoints using Red Hat Ansible Automation Platform (AAP) and Red Hat OpenShift Service on AWS (ROSA). Focuses on decoupling Control and Execution Planes, and leveraging automatic cluster autoscaling to optimize compute costs.
2. **[Blog 2 – Introducing the AWS WAF Traffic Overview Dashboard](3.2-WAFDashboard/)**
   * *Overview:* Introduces the free, out-of-the-box traffic overview dashboard in AWS WAF. Explains how security administrators can monitor baseline traffic patterns, analyze sudden traffic spikes, and optimize AWS WAF Bot Control rules to protect application endpoints.
3. **[Blog 3 – Separating Read and Write Operations in Serverless APIs](3.3-ServerlessCQRS/)**
   * *Overview:* Details trade-offs between Single-Purpose Lambda functions and Monolithic Lambdas. Proposes a pragmatic middle-ground pattern (Read/Write Separation) to optimize deployment package sizes, reduce cold start occurrences, and establish an evolutionary path toward CQRS architectures.
